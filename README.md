## Usage
### start server 
$ go run main.go

$ curl http://localhost:8080/response

### set json
- set json you wanting mock in fixtures
- json file rename 'response.json' 


## Setup
$ brew install go  

$ vim ~/.bash_profile  
```
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin
```
$ source ~/.bash_profile  
  
$ cd ~  
$ mkdir go go/src go/bin go/pkg
