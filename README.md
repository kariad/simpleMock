## Setup
$ brew install go  

$ vim ~/.bash_profile  
```
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin
```
$ source ~/.bash_profile  
  
$ cd ~  
$ mkdir go go/src go/bin go/pkg
